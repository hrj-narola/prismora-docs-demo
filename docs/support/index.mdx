# Prismora Support

This section provides a standard, end-to-end guide to the Prismora workflow, from account setup through testing, integrations, and collaborating with project members.

:::tip Start here
Use the workflow order shown below. Each page links forward to the next step so you can follow the same path as a real user.
:::

## What you can do here

- Register and login
- Create and configure projects
- Upload resources, analyze them, and finish setup
- Generate test cases and run tests
- View test results (including common action buttons)
- Use multiple test case input sources (code changes, requirements, design files, and free-form descriptions)
- Connect integrations (GitHub, Asana, etc.)
- Use the Knowledge Base feature
- Manage project members and invites
