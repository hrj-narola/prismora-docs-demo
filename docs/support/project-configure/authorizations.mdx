# Authorizations

The **Authorizations** tab allows you to manage connected integrations for your specific project. These integrations allow Prismora to access secure project resources and generate comprehensive test coverage tailored to your workflows.

## Managing Connections

If your project requires external read access (e.g., Jira, GitHub, or secure endpoints), the necessary connectors will be presented here based on the data evaluated during analysis.

- **No Connections Required**: If your project boundaries don't require external links, the platform will display a success state confirming that you are good to go.
- **Integration Info**: Use this section to proactively connect additional services to enhance test coverage and enable advanced automated workflows.

You can instruct the bot on the left to walk you through specific authorization processes if an integration requires nuanced token access.
