# Project Configuration

The **Project Configuration** portal is where you govern the scope, context, and operational bounds of your Prismora project. 

The configuration interface is incredibly intuitive: the **Knowledge Assistant** chatbot remains constantly available on the left side of your screen to answer questions or process new uploads, while the right side of the screen updates depending on which configuration tab you select.

## Configuration Tabs

Prismora organizes project settings into 7 distinct tabs:

1. [Project Overview](/docs/support/project-configure/project-overview)
2. [Authorizations](/docs/support/project-configure/authorizations)
3. [Test Data](/docs/support/project-configure/test-data)
4. [Knowledge Base](/docs/support/project-configure/knowledge-base)
5. [Project Members](/docs/support/project-configure/project-members)
6. [Notifications](/docs/support/project-configure/notifications)
7. [Settings](/docs/support/project-configure/settings)

Explore each section to fully customize your project’s automated testing environment.
