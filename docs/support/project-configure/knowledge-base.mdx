# Knowledge Base

Your **Knowledge Base** is the living repository of context that drives Prismora's AI test generation. 

## Uploading and Managing Context

While you can always converse with the Knowledge Assistant on the left side of the screen, the right side displays the concrete files and context notes you've aggregated:

- **Upload Resources**: Click the prominent **Upload** button to introduce new documentation, requirement files, or design mockups.
- **Files**: Review the distinct files directly associated with the project.
- **Notes & Context**: View text snippets, URL scraping results, or typed context (e.g., explicit website sub-pages or operational guidelines).
- **Search**: Use the *Search files and notes...* bar to quickly locate older context items as your project scales.

If you ever need to refine a distinct user flow, simply upload new technical documentation here and the AI will recalibrate your tests automatically.
