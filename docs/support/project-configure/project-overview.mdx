# Project Overview

The **Project Overview** tab serves as the primary dashboard for viewing the AI’s high-level understanding of your application. 

The left pane retains your conversational history with the Knowledge Assistant, while the right pane displays the structured summary and analysis established during project onboarding.

## Key Sections

You can view and directly update the following critical parameters to ensure the AI creates perfectly scoped test cases:

- **Description**: A high-level explanation of your project's primary purpose.
- **Tech Stack**: The underlying technologies powering your application (e.g., HTML, CorePHP, MySQL, React). 
- **Key Features**: The primary functional pillars of your application.
- **Critical User Flows**: The vital pathways that users navigate (e.g., checkout, login, registration).
- **Estimated Test Coverage**: A top-level metric projecting the testing depth based on the provided context.

To edit any of these parameters, simply click the **Edit** (pencil) icon located in the respective card and provide the updated information.
