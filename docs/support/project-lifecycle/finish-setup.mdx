# Finish Setup & Generate Tests

Once your project summary and architecture are meticulously refined using the Chatbot, you are ready to conclude the onboarding phase.

## Finalizing the Project

- A dedicated banner will appear below your analysis indicating the platform is **Ready to Generate Tests**.
- Reviewing the updated analysis confirms the onboarding lifecycle is successfully completed.

## Generating Tests

- Click the **Continue** button located on the banner. 
- Prismora will gracefully transition from onboarding to actively creating comprehensive test cases explicitly tailored to your newly finalized project analysis.

By successfully generating tests, your project is fully onboarded.
