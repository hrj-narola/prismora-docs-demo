# Finish Setup

**Finish Setup** finalizes the configuration based on the analysis output and enables the testing workflow (test case generation and test runs).

## What you should verify

- The inputs/resources selected for testing are correct.
- The scope matches your goal (for example: the right feature/release boundaries).
- Any configuration options shown in your UI are set to your intended behavior.

## Steps

1. Open the project setup screen.
2. Select **Finish Setup**.
3. Review the final configuration:
   - Confirm the input set that will drive test generation.
   - Check any generated/derived context (if displayed in your UI).
4. Click **Confirm** / **Finish**.

## Expected result

- The project is ready for the testing workflow.
- Controls like **Number of test cases**, **Generate Test Cases**, and **Test Run** become available (or get enabled).

## Troubleshooting

- If testing controls are missing/disabled, go back and confirm:
  - Analyze completed successfully
  - Finish Setup was confirmed
  - Your permissions allow testing actions
