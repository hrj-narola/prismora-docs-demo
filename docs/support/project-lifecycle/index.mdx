# Project Lifecycle

Use this section to understand how to onboard your Prismora project efficiently. Setting up a new project and configuring its testing scope is intelligently driven by our conversational AI Assistant.

:::tip Follow the workflow in order
Each step builds upon the previous one. The AI Chatbot makes onboarding dynamic, so if something looks wrong later, you can always go back and instruct the chatbot to make adjustments before generating tests.
:::

## Steps

1. [Project Initialization](/docs/support/project-lifecycle/chatbot-interaction)
2. [Upload Resource](/docs/support/project-lifecycle/upload-resource)
3. [Analyze](/docs/support/project-lifecycle/analyze)
4. [Finish Setup](/docs/support/project-lifecycle/finish-setup)

## Quick checklist

- Your initial project context is shared via the chatbot.
- Your resources (design files, requirements, external URLs) are successfully uploaded or explained.
- The AI's comprehensive analysis (Project Summary, Tech Stack, Authorizations) completes successfully and has been thoroughly reviewed.
- Final setup is confirmed via the **Continue** button, and testing generation safely begins.
