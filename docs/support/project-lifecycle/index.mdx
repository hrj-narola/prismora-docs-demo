# Project Lifecycle

Use this section to create a Prismora project, upload your inputs, analyze them, and finish setup so you can start generating and running test cases.

:::tip Follow the workflow in order
Each step prepares the next one. If something looks wrong later, re-check inputs and re-run Analyze.
:::

## Steps

1. [Create Project](/docs/support/project-lifecycle/create-project)
2. [Upload Resource](/docs/support/project-lifecycle/upload-resource)
3. [Analyze](/docs/support/project-lifecycle/analyze)
4. [Finish Setup](/docs/support/project-lifecycle/finish-setup)

## Quick checklist

- Your project is created.
- Your resources are uploaded with the correct resource type.
- Analysis completes successfully (or you understand what it produced).
- Finish Setup is confirmed, and testing controls become available.
