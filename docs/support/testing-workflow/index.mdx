# Testing Workflow

Welcome to the Prismora Testing Workflow. Once your project onboarding is completed and your context is fully analyzed, Prismora automatically generates comprehensive test suites for your application.

This section explains the end-to-end testing lifecycle: viewing your generated test cases, reviewing and editing detailed test steps, launching test runs, and evaluating AI-driven insights from the results.

## Steps

1. [Test Cases List](/docs/support/testing-workflow/test-cases)
2. [Test Case Details & Editing](/docs/support/testing-workflow/test-case-details)
3. [Live Console](/docs/support/testing-workflow/live-console)
4. [Test Results](/docs/support/testing-workflow/test-results)

## Quick Checklist

- Your onboarding is completed and test generation has finished.
- You have reviewed the generated test cases to verify their scope.
- You have selectively run tests and monitored them via the Live Console.
- You have analyzed the run history and AI Insights on the results dashboard.
