# Test Run

**Test Run** executes the generated test cases in your selected environment (or with your selected configuration).

:::caution When to run
Run tests only after you have:
- the project ready (Finish Setup done), and
- generated test cases (so you know what will be executed).
:::

## What to do

1. Choose **Test Run**.
2. Confirm run settings (for example: environment and/or which test set to execute).
3. Start the run.
4. Wait for execution to complete.

## Troubleshooting

- If the run cannot start: confirm project setup is complete and you have the required permissions.
- If many tests fail unexpectedly: verify you generated tests using the correct inputs and scope.
